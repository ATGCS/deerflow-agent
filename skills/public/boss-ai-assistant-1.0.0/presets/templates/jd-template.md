# 职位描述模板

## 技术类岗位JD模板

### 前端工程师

```
【岗位职责】
1. 负责公司Web产品的前端开发工作，确保代码质量和用户体验
2. 参与产品需求讨论，提供技术可行性建议
3. 优化前端性能，提升页面加载速度和交互体验
4. 与设计师、后端工程师紧密配合，完成产品功能开发
5. 持续学习新技术，推动团队技术升级

【任职要求】
1. 本科及以上学历，计算机相关专业优先
2. X年以上前端开发经验，精通HTML5/CSS3/JavaScript
3. 熟练掌握Vue/React/Angular等主流框架至少一种
4. 熟悉前端工程化工具（Webpack/Vite等）
5. 了解HTTP协议、浏览器渲染原理
6. 具备良好的代码规范和团队协作能力

【加分项】
- 有大型项目开发经验
- 熟悉Node.js后端开发
- 有移动端H5或小程序开发经验
- 参与过开源项目

【薪资福利】
- 薪资范围：XXK-XXK（根据能力面议）
- 五险一金，带薪年假
- 弹性工作制，扁平化管理
- 技术分享会，培训学习基金
- 节日福利，团队建设活动
```

### Java工程师

```
【岗位职责】
1. 负责公司核心业务系统的后端开发与维护
2. 参与系统架构设计，解决高并发、高可用技术难题
3. 编写高质量代码，进行Code Review
4. 优化系统性能，保障服务稳定性
5. 参与技术选型和技术方案制定

【任职要求】
1. 本科及以上学历，计算机相关专业
2. X年以上Java开发经验，扎实的Java基础
3. 精通Spring Boot/Spring Cloud微服务架构
4. 熟悉MySQL/Redis/MQ等中间件
5. 了解分布式系统原理，有实际项目经验
6. 具备良好的问题分析和解决能力

【加分项】
- 有互联网大厂工作经验
- 熟悉K8s/Docker容器化部署
- 有大数据处理经验
- 具备团队管理经验

【薪资福利】
- 薪资范围：XXK-XXK（14薪）
- 技术大牛带队，成长空间大
- 完善的晋升体系和培训机制
```

## 产品/运营类岗位JD模板

### 产品经理

```
【岗位职责】
1. 负责产品规划、需求分析和功能设计
2. 撰写PRD文档，推动产品从0到1落地
3. 协调设计、开发、测试资源，把控项目进度
4. 收集用户反馈，持续优化产品体验
5. 分析竞品动态，制定产品竞争策略

【任职要求】
1. 本科及以上学历，3年以上产品经理经验
2. 具备完整的产品设计能力，熟练使用Axure/Sketch
3. 逻辑思维清晰，具备良好的数据分析能力
4. 优秀的沟通协调能力，能推动跨部门合作
5. 对用户体验有深刻理解，有C端产品经验优先

【加分项】
- 有成功上线的产品案例
- 技术背景或懂基础技术原理
- 有创业经历或0-1产品经验

【薪资福利】
- 薪资范围：XXK-XXK
- 产品决策权大，发挥空间大
- 与优秀团队共事，快速成长
```

## 使用说明

1. 根据实际岗位调整"X年经验"和"薪资范围"
2. 补充公司特色福利和优势
3. 技术栈根据实际要求修改
4. 可在"加分项"中突出公司看重的特殊能力
