# 沟通话术模板

## 面试邀约话术

### 初次邀约（电话/短信）

```
您好，请问是[候选人姓名]吗？我是[公司名称]的HR[你的名字]。

在BOSS直聘上看到您的简历，觉得您的经历和我们[岗位名称]岗位很匹配，想邀请您来面试聊聊。

【岗位亮点】
- [一句话突出岗位优势，如：核心业务部门/技术大牛带队/薪资有竞争力]
- [可选：公司规模、发展阶段、团队氛围等]

【面试安排】
- 时间：[具体日期] [上午/下午] [时间段]
- 地点：[公司地址]
- 时长：约[X]分钟
- 形式：[现场面试/视频面试]

您这个时间方便吗？稍后我会把详细地址和注意事项发到您手机上。
```

### 面试邀请邮件

```
主题：【面试邀请】[公司名称] - [岗位名称] - [候选人姓名]

尊敬的[候选人姓名]，您好！

感谢您对[公司名称]的关注。经过初步筛选，我们诚挚邀请您参加面试。

【面试信息】
岗位名称：[岗位名称]
面试时间：[日期] [时间]
面试地点：[详细地址]
面试形式：[现场/视频]
预计时长：[X]分钟

【面试流程】
1. HR初试（约20分钟）
2. 部门负责人复试（约30分钟）
3. [可选：技术笔试/上机测试]

【携带材料】
- 个人简历1份
- 身份证原件
- [其他要求，如：作品集、学历证书等]

【联系方式】
HR：[姓名]
电话：[手机号]
微信：[微信号]

期待与您的见面！

[公司名称]
[日期]
```

## 面试提醒

### 面试前一天提醒

```
[候选人姓名]您好，我是[公司名称]HR[你的名字]。

提醒您明天[时间]的面试：

📍 地址：[详细地址]
🚇 交通：[地铁/公交路线]
⏰ 时间：[具体时间]
📱 联系人：[姓名] [电话]

【温馨提示】
- 请携带身份证和个人简历
- 建议提前10-15分钟到达
- [天气提醒/停车信息]

如有变动请提前告知，期待明天见到您！
```

## 面试结果反馈

### 面试通过通知

```
[候选人姓名]您好！

很高兴通知您，您已通过[岗位名称]的面试！🎉

【下一步】
我们将在[X]个工作日内向您发送正式offer，包含：
- 薪资待遇详情
- 入职时间安排
- 入职材料清单

【薪资初步沟通】
您的期望薪资是[金额]，综合您的经验和面试表现，我们可以给到[金额]（[13/14]薪），您看是否可以接受？

期待您的加入！如有任何问题随时联系我。

[HR姓名]
[电话]
[日期]
```

### 面试未通过通知（委婉版）

```
[候选人姓名]您好，

感谢您抽出宝贵时间参加[公司名称][岗位名称]的面试。

经过综合评估，我们认为您在[某方面]非常优秀，但目前岗位需求与您的经验匹配度方面存在一些差异。

【建议】
- 您的[优势领域]很强，建议可以继续深耕
- [可选：推荐其他更适合的岗位]

我们会将您的简历保留在人才库，如有更合适的岗位会第一时间联系您。

祝您早日找到满意的工作机会！

[公司名称]HR团队
[日期]
```

### 面试未通过通知（简洁版）

```
[候选人姓名]您好，

感谢您参加[公司名称]的面试。

经过慎重考虑，本次岗位与您的匹配度不够理想，暂时无法达成合作。

我们会保留您的简历，期待未来有机会再合作。

祝您职业发展顺利！
```

## Offer沟通

### 发送Offer前电话沟通

```
[候选人姓名]您好，我是[公司名称]HR[你的名字]。

有个好消息告诉您，您已通过面试，我们准备给您发offer了！

【offer概况】
- 岗位：[岗位名称]
- 薪资：[金额]元/月（[13/14]薪）
- 入职时间：[日期]
- 试用期：[X]个月，薪资[比例]%

【福利补充】
- 五险一金按[比例/基数]缴纳
- 带薪年假[X]天
- [其他福利]

您对这个方案有什么想法吗？如果没有问题，我今天就发正式offer到您邮箱。
```

## 入职跟进

### 入职前一周提醒

```
[候选人姓名]您好！

距离您入职还有一周，跟您确认几件事：

✅ 离职流程进展如何？能按时入职吗？
✅ 入职材料准备齐全了吗？
✅ 对offer还有什么疑问吗？

【入职当天】
- 时间：[时间]（建议提前15分钟）
- 地点：[地址]
- 联系人：[姓名] [电话]
- 着装要求：[要求]

【第一天安排】
- 办理入职手续
- 熟悉办公环境
- 部门介绍和团队认识
- 工作设备领取

有任何问题随时联系我！
```

```
[候选人姓名]您好！欢迎加入[公司名称]！🎉

我是HR[你的名字]，今天由我帮您办理入职。

【今日流程】

【午餐安排】
中午[时间]，我会带您和团队一起吃个饭，熟悉一下同事。

再次欢迎加入，有任何问题随时找我！
```

## 薪资谈判

### 应对薪资要求过高

```
我理解您的期望，您的经验和能力确实很有竞争力。

【我们的考虑】
- 公司目前的薪资体系是[说明]
- 这个岗位目前的预算范围是[范围]
- 我们可以提供的是[金额]，包括[构成说明]

【其他补偿】
虽然基础薪资暂时达不到您的期望，但我们还有：
- 年终奖[X]个月
- 项目奖金/绩效奖金
- 期权/股权激励[如有]
- 快速晋升通道

综合下来，年收入大约在[金额]左右。而且公司发展很快，半年到一年有调薪机会。

您看这样的方案能接受吗？
```

### 挽留优秀候选人

```
[候选人姓名]，我知道您可能还在考虑其他机会。

【选择我们的理由】
- [业务优势：如核心业务、快速增长]
- [团队优势：如技术大牛、学习氛围]
- [发展优势：如晋升快、项目机会多]
- [薪资优势：如年终奖高、福利好]

【特别说明】
我们真的很看重您，[直属领导]对您的面试评价很高。

如果您能接受offer，我可以申请：
- [特殊福利，如：签字费、提前转正等]

希望您能给我们一个一起共事的机会！
```
