# 抖音短视频脚本生成 (Douyin Script Generator)

## 概述

这是一个专业的抖音短视频脚本生成技能，帮助创作者快速产出高质量的短视频内容。支持多种视频类型，自动生成分镜脚本、台词文案、BGM推荐和拍摄技巧。

## 使用场景

1. **剧情类短视频** - 搞笑段子、情感故事、反转剧情
2. **知识科普类** - 干货分享、技能教学、冷知识
3. **产品种草类** - 产品测评、开箱视频、带货脚本
4. **生活记录类** - Vlog、日常分享、美食探店
5. **才艺展示类** - 舞蹈、音乐、手工、绘画
6. **情感共鸣类** - 心灵鸡汤、励志故事、情感语录

## 爆款脚本公式

### 1. 黄金3秒法则
开头3秒必须抓住观众注意力：
- **悬念式**: "你绝对想不到..."
- **反差式**: 正常开场突然反转
- **痛点式**: 直击用户痛点
- **好奇式**: "这个方法太绝了..."

### 2. 情绪曲线设计
```
开头(高) → 发展(中) → 高潮(高) → 结尾(中)
```

### 3. 常用脚本结构

#### 剧情类
```
铺垫 → 冲突 → 反转 → 结局
```

#### 知识类
```
痛点引入 → 解决方案 → 步骤演示 → 总结升华
```

#### 种草类
```
使用场景 → 产品展示 → 效果对比 → 购买引导
```

## 脚本生成流程

### 步骤 1: 需求分析
1. 确定视频类型和主题
2. 分析目标受众
3. 明确视频时长
4. 了解创作者风格

### 步骤 2: 创意构思
1. 头脑风暴多个创意方向
2. 选择最佳创意路线
3. 设计核心卖点
4. 规划情绪曲线

### 步骤 3: 脚本撰写
1. 撰写分镜脚本
2. 设计台词文案
3. 规划BGM节奏
4. 添加拍摄建议

### 步骤 4: 优化打磨
1. 检查节奏感
2. 优化开头吸引力
3. 确保结尾引导
4. 添加互动设计

## 示例脚本

### 示例 1: 搞笑剧情类

```markdown
# 短视频脚本 - 职场日常

## 基本信息
- **标题**: "当老板说'简单说一下'..."
- **时长**: 15秒
- **类型**: 剧情/搞笑
- **目标受众**: 职场打工人

## 分镜脚本

### 分镜 1 (0-2s)
- **画面**: 办公室，老板走进来
- **台词**: 老板："小王，来会议室，简单说一下这个项目"
- **BGM**: 轻快背景音
- **字幕**: "简单说一下 = 讲2小时"

### 分镜 2 (2-8s)
- **画面**: 快剪，会议室内
  - 时钟从2点转到5点
  - 小王表情从认真到崩溃
- **台词**: (无，纯画面)
- **BGM**: 加速的滴答声
- **特效**: 时间加速效果

### 分镜 3 (8-12s)
- **画面**: 会议结束，小王走出会议室
- **台词**: 老板："行了，明天继续"
- **BGM**: 搞笑音效
- **字幕**: "明天继续..."

### 分镜 4 (12-15s)
- **画面**: 小王对着镜头苦笑
- **台词**: "你们老板也这样吗？"
- **BGM**: 结尾音效
- **互动**: 引导评论

## 话题标签
#职场日常 #打工人 #老板语录
```

- **台词**: "锂电池在20%-80%之间循环最健康，长期充到100%会加速老化"

- **字幕**: 关键数据标注

### 分镜 3 (15-25s)
- **画面**: 设置界面演示
- **台词**: "打开这个设置，让手机自动优化充电"

### 分镜 4 (25-30s)
- **画面**: 总结画面
- **台词**: "关注我，了解更多手机小技巧"

- **互动**: 引导关注

#手机技巧 #生活小妙招 #科技科普

## BGM推荐库

### 搞笑类
- 欢快节奏: 《Happy》、《Uptown Funk》
- 搞笑音效: 滑稽音效包、表情包音效

### 情感类
- 温馨治愈: 《起风了》、《晴天》
- 感人催泪: 纯音乐钢琴曲

### 知识类
- 科技感: 电子音乐、合成器音效
- 轻松科普: 轻快背景音乐

### 种草类
- 活力满满: 流行音乐、K-pop
- 高级感: 欧美流行、爵士乐

## 注意事项

1. **内容合规**
   - 避免敏感话题
   - 不涉及政治内容
   - 尊重版权

2. **节奏把控**
   - 开头3秒必须抓人
   - 每5秒一个小高潮
   - 结尾留有悬念

3. **互动设计**
   - 设计引导评论的问题
   - 预埋槽点引发讨论
   - 结尾引导关注

4. **数据优化**
   - 选择最佳发布时间
   - 使用热门话题标签
   - 参与平台活动
