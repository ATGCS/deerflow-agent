## 🐛 已知问题

### 问题 1: 首次搜索较慢

**现象**: 首次搜索耗时 30-40 秒  
**原因**: 百度搜索工具使用 Playwright 真浏览器，需要启动浏览器并加载页面  
**解决**: 
- 设置 `DEERFLOW_BAIDU_PAGE_CONTENT_CHARS=0` 禁用内容提取
- 使用缓存避免重复搜索

### 问题 2: langchain 依赖

**现象**: 未安装 langchain 时导入失败  
**原因**: __init__.py 默认导入 langchain  
**解决**: 
- 已修复为可选依赖
- 不安装 langchain 仍可使用 Python API
