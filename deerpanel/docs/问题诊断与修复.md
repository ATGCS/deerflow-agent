## 🐛 问题描述

### 问题 1: 页面不显示任务

**现象**:
- 任务已创建（ID: b2fc0406）
- 任务已执行并写入文件
- 但前端页面没有任何显示

**原因分析**:
1. ❌ 前端未集成任务进度可视化系统
2. ❌ 没有连接 SSE 事件流接收实时更新
3. ❌ 聊天页面没有嵌入任务仪表板
4. ❌ 没有调用任务相关的 API

**影响**: 用户无法在页面中看到任务执行进度和结果

---

### 问题 2: Lead Agent 行为不连贯

**现象**:
- Lead Agent 应该规划和分配任务
- 但实际上 Lead Agent 自己执行了文件写入
- 没有正确分配给子代理

**原因分析**:
1. ⚠️ Lead Agent 的职责定义不清晰
2. ⚠️ 任务分配机制有问题
3. ⚠️ 可能缺少子代理调用逻辑

**影响**: 多智能体协作的优势无法体现

## 🛠️ 修复方案

### 修复 1: 前端集成（必须）

#### 1.1 在 chat.js 中添加任务支持

修改 `deerpanel/src/pages/chat.js`:

```javascript
// 在文件顶部添加导入
import { tasksAPI } from '../lib/api-client.js'
import { EventStreamManager } from '../lib/event-stream.js'
import { FloatingTaskPanel } from '../components/FloatingTaskPanel.js'
import { EmbeddedTaskDashboard } from '../components/EmbeddedTaskDashboard.js'
import { StateRestorationManager } from '../lib/state-persistence.js'

// 在 render 函数中添加任务仪表板容器
export async function render() {
  const page = document.createElement('div')
  page.className = 'page'
  
  // ... 现有代码 ...
  
  // ========== 新增：任务系统集成 ==========
  
  // 1. 创建嵌入式仪表板容器
  const messageList = page.querySelector('#message-list')
  const dashboardContainer = document.createElement('div')
  dashboardContainer.id = 'embedded-dashboard-container'
  dashboardContainer.style.margin = '16px 0'
  
  if (messageList.firstChild) {
    messageList.insertBefore(dashboardContainer, messageList.firstChild)
  }
  
  // 2. 初始化浮动面板
  window.__floatingPanel = new FloatingTaskPanel()
  
  // 3. 状态恢复
  const restorationManager = new StateRestorationManager()
  await restorationManager.restore({
    onCacheRestore: (data) => {
      console.log('[ChatPage] 从缓存恢复:', data)
      if (data.panelState && data.panelState.isOpen) {
        window.__floatingPanel.open()
      }
    },
    onAPIRestore: async (data) => {
      console.log('[ChatPage] 从 API 恢复:', data)
      if (data.tasks && data.tasks.length >= 2) {
        // 显示嵌入式仪表板
        const container = document.getElementById('embedded-dashboard-container')
        if (container) {
          const dashboard = new EmbeddedTaskDashboard(container)
          await dashboard.loadTasks()
          dashboard.startAutoRefresh()
          window.__embeddedDashboard = dashboard
        }
      }
    },
    onSSEConnect: () => {
      console.log('[ChatPage] SSE 已连接')
      // SSE 会自动更新
    }
  })
  
  // ======================================
  
  return page
}
```

### 修复 2: Lead Agent 行为修正

#### 2.1 检查 Lead Agent 实现

查看 Lead Agent 的代码，确保它：
1. 先规划任务（创建主任务）
2. 分解为子任务
3. 分配给不同的子代理
4. 收集结果并汇总

**不应该**:
- ❌ Lead Agent 直接执行具体操作
- ❌ 跳过子任务分配

#### 2.2 修正任务分配逻辑

如果 Lead Agent 直接执行了操作，检查：

```python
# backend/.../lead_agent.py 或类似文件

# 错误的实现：
async def execute_task(self, task):
    # ❌ Lead Agent 自己执行
    result = await self.write_file(...)
    return result

# 正确的实现：
async def execute_task(self, task):
    # ✅ 创建子任务并分配
    subtask = await self.create_subtask(
        name="写入文件",
        agent="coder",  # 分配给 coder 代理
        description="..."
    )
    # 等待子任务完成
    result = await self.wait_for_subtask(subtask.id)
    return result
```
