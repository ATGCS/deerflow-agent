## 🐛 常见问题排查

### 问题 1: 后端连接失败

**错误**: `Failed to fetch`

**解决**:
```bash
# 1. 检查后端是否运行
curl http://localhost:8000/api/tasks

# 2. 检查端口是否被占用
netstat -ano | findstr :8000

# 3. 重启后端服务
cd backend
uvicorn app.gateway.main:app --reload --port 8001
```

### 问题 2: CORS 错误

**错误**: `Access to fetch at 'http://localhost:8000' from origin 'http://localhost:5173' has been blocked by CORS policy`

**解决**:
```python
# 在 backend/main.py 中添加 CORS 配置
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

### 问题 3: 前端样式未加载

**错误**: 页面显示不正常，没有样式

**解决**:
```bash
# 1. 清除缓存
# Ctrl+Shift+Delete -> 清除缓存和 Cookie

# 2. 强制刷新
# Ctrl+F5

# 3. 检查 CSS 文件路径
# 查看 Network 标签，确认 components.css 加载成功
```

### 问题 4: 浮动面板无法拖动

**错误**: 面板打开但无法拖动

**解决**:
```javascript
// 在控制台检查
const panel = document.querySelector('.floating-task-panel')
console.log('面板元素:', panel)
console.log('面板位置:', panel?.style.left, panel?.style.top)

// 如果面板不存在，重新打开
// 按 Ctrl+T 或执行:
await window.__floatingPanel.open()
```

### 问题 5: SSE 连接失败

**错误**: `EventSource connection failed`

**解决**:
```javascript
// 在控制台检查
console.log('EventSource 状态:', EventSource)

// 手动测试连接
const es = new EventSource('http://localhost:8000/api/events/projects/test/stream')
es.onopen = () => console.log('✅ SSE 连接成功')
es.onerror = (e) => console.error('❌ SSE 连接失败:', e)
```
