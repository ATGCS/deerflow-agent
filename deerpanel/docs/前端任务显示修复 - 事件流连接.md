# 前端任务显示问题修复 - 事件流连接修复

## 🐛 问题描述

**现象**: 主智能体已经规划了任务，但前端页面不显示任务

**原因**: 事件流（SSE）连接逻辑有误，导致无法接收任务创建事件

## 🔧 修复内容

### 文件位置
`d:\github\deerflaw\deerpanel\src\pages\chat.js`

### 问题分析

#### 原代码问题（第 4075-4142 行）

```javascript
// ❌ 错误 1: 错误地实例化 EventStreamManager
window.__taskEventStream = new EventStreamManager((event) => {
  // 回调函数不会被调用
})

// ❌ 错误 2: connect() 方法缺少项目 ID 参数
```

**问题根源**:
1. `EventStreamManager` 是单例模式，不应该用 `new` 创建实例
2. `connect()` 方法需要项目 ID 参数
3. 事件监听器绑定逻辑混乱，导致无法正确接收事件

### 修复后代码

```javascript
async function initTaskVisualization(page) {
  try {
    // 1. 初始化嵌入式任务仪表板
    const dashboardContainer = page.querySelector('#embedded-task-dashboard')
    if (dashboardContainer) {
      window.__embeddedTaskDashboard = new EmbeddedTaskDashboard(dashboardContainer)
      console.log('[Chat] 嵌入式任务仪表板已初始化')
    }
    
    // 2. 初始化浮动任务面板
    window.__floatingPanel = new FloatingTaskPanel()
    console.log('[Chat] 浮动任务面板已初始化')
    
    // 3. 获取或创建项目事件流（修复核心）
    const eventStreamManager = EventStreamManager.getInstance()  // ✅ 使用单例
    const projectId = 'main'  // ✅ 指定项目 ID
    const taskEventStream = eventStreamManager.getStream(projectId)
    
    // 4. 添加事件监听器（修复事件绑定）
    taskEventStream.on(EventTypes.TASK_CREATED, async (data) => {
      console.log('[Chat] 检测到新任务创建:', data)
      setTimeout(async () => {
        try {
          const taskId = data.taskId || data.id
          if (taskId) {
            const task = await tasksAPI.getTask(taskId)
            if (task && window.__embeddedTaskDashboard) {
              window.__embeddedTaskDashboard.showTask(task)
              console.log('[Chat] 已显示新任务:', taskId)
            }
          }
        } catch (err) {
          console.warn('[Chat] 获取任务详情失败:', err)
        }
      }, 500)
    })
    
    // 监听任务进度更新
    taskEventStream.on(EventTypes.TASK_PROGRESS, (data) => {
      console.log('[Chat] 收到任务进度更新:', data)
      if (window.__embeddedTaskDashboard) {
        window.__embeddedTaskDashboard.handleEvent({
          type: EventTypes.TASK_PROGRESS,
          data: data
        })
      }
      if (window.__floatingPanel) {
        window.__floatingPanel.handleEvent({
          type: EventTypes.TASK_PROGRESS,
          data: data
        })
      }
    })
    
    // 监听任务完成事件
    taskEventStream.on(EventTypes.TASK_COMPLETED, (data) => {
      console.log('[Chat] 任务完成:', data)
      if (window.__embeddedTaskDashboard) {
        window.__embeddedTaskDashboard.handleEvent({
          type: EventTypes.TASK_COMPLETED,
          data: data
        })
      }
      if (window.__floatingPanel) {
        window.__floatingPanel.handleEvent({
          type: EventTypes.TASK_COMPLETED,
          data: data
        })
      }
    })
    
    // 5. 连接事件流
    taskEventStream.connect()  // ✅ 直接调用 connect()
    console.log('[Chat] SSE 事件流已连接')
    
    // 6. 状态恢复
    const restorationManager = new StateRestorationManager()
    const activeTask = await restorationManager.restoreActiveTask()
    
    if (activeTask && window.__embeddedTaskDashboard) {
      window.__embeddedTaskDashboard.showTask(activeTask)
      console.log('[Chat] 已恢复活动任务:', activeTask.id)
    }
    
    console.log('[Chat] 任务进度可视化系统集成完成')
  } catch (err) {
    console.error('[Chat] 任务系统集成失败:', err)
  }
}
```

## 🔍 技术细节

### EventStreamManager 单例模式

```javascript
// event-stream.js 第 283-301 行
export class EventStreamManager {
  static instance = null
  
  static getInstance() {
    if (!EventStreamManager.instance) {
      EventStreamManager.instance = new EventStreamManager()
    }
    return EventStreamManager.instance
  }
  
  getStream(projectId) {
    if (!this.streams.has(projectId)) {
      const stream = new TaskEventStream(projectId)
      this.streams.set(projectId, stream)
    }
    return this.streams.get(projectId)
  }
}
```

**关键点**:
1. 必须使用 `getInstance()` 获取单例
2. 通过 `getStream(projectId)` 获取项目专属的 TaskEventStream
3. TaskEventStream 需要项目 ID 才能连接正确的 SSE 端点

### TaskEventStream 连接逻辑

```javascript
// event-stream.js 第 35-51 行
connect() {
  const url = `${API_BASE}/api/events/projects/${this.projectId}/stream`
  console.log('[EventStream] Connecting to:', url)
  
  this.eventSource = new EventSource(url)
  this.setupEventHandlers()
}
```

**关键点**:
- URL 中包含项目 ID：`/api/events/projects/${projectId}/stream`
- 如果 `projectId` 未指定，URL 会是 `/api/projects/undefined/stream` → 连接失败
